# 💊 Smart Pharmacy Assistant

نظام ذكي لاقتراح الأدوية باستخدام الذكاء الاصطناعي، مع بيانات حقيقية من السوق المصري.

## 🚀 المميزات
- اقتراح دواء بناءً على بيانات المريض (العمر، الضغط، الكوليسترول، نسبة الصوديوم/البوتاسيوم)
- قاعدة بيانات تحتوي على أدوية حقيقية مع أسعار، بدائل، وتحذيرات
- واجهة مستخدم تفاعلية بـ Streamlit
- إمكانية البحث عن دواء
- مشروع كامل مناسب للـ CV أو مشروع التخرج

## 🛠️ التقنيات المستخدمة
- Python
- Streamlit
- Scikit-learn (Decision Tree)
- Pandas
